## 🚀 如何使用

### ① 浏览演示

**推荐入口：[`presentation.html`](./presentation.html)**

由于单文件 PPT 体积过大（22 页 + 8 张 Chart.js 图表），项目采用「骨架 + 片段」的方式组织：

- `presentation.html` — 入口骨架（自带 Tailwind/Chart.js/思源字体 CDN、键盘/滚轮翻页、图表渲染逻辑）
- `slides/s1.html` ～ `slides/s22.html` — 22 个独立的页面片段，运行时由骨架并发加载注入

**使用方法**：
```bash
# 方法 A：直接双击 presentation.html（部分浏览器需启动本地服务以允许 fetch 加载片段）
# 方法 B：本地起一个静态服务器（推荐）
python3 -m http.server 8000
# 然后浏览器访问  http://localhost:8000/presentation.html
```
